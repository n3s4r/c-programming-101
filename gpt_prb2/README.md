# C Programming Problem: Reverse an Array (In-Place)

## Problem Description

Write a **C program** that reverses the elements of an array **without using another array**.

Your program should perform the following steps:

1. Ask the user to enter the **number of elements (n)**.
2. Read **n integers** from the user and store them in an array.
3. Reverse the array **in-place** (meaning you must modify the original array instead of creating a second array).
4. Print the **reversed array**.

---

## Constraints

- Do **not use another array**.
- The reversal must happen **inside the original array**.
- Use **loops and variables only**.
- Assume the maximum number of elements is **100**.
