# C Programming Problem: Second Largest Number

## Problem Description

Write a **C program** that finds the **second largest number** in a list of integers.

Your program should:

1. Take an integer **n** from the user (number of elements).
2. Take **n integers** as input and store them in an array.
3. Determine the **second largest number** in the array.
4. Print the result.

If a second largest number **does not exist** (for example, when all numbers are the same), the program should print:

Second largest number does not exist


---

## Constraints

- Do **not sort the array**.
- Use **loops and conditional statements** to determine the second largest number.
- The program should correctly handle **negative numbers**.

---

## Example 1

**Input**
Enter number of elements: 5
Enter the elements: 10 5 8 20 15


**Output**
Second largest number: 15
---

## Example 2

**Input**
Enter number of elements: 4
Enter the elements: 7 7 7 7


**Output**
Second largest number does not exist


---

## Expected Concepts Used

- Arrays
- Loops (`for`)
- Conditional statements (`if`, `else if`)
- Basic input/output (`scanf`, `printf`)
