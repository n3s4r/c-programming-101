# Problem: Reverse Words in a String

## 📝 Description
Given a string, reverse the order of the words **in-place**. 

- A word is a sequence of characters separated by spaces.
- You must modify the original string without using extra space for another string.

## 💡 Example
**Input:** `"hello world"`  
**Output:** `"world hello"`

**Input:** `"The quick brown fox"`  
**Output:** `"fox brown quick The"`

## ⚙️ Requirements
1. Implement the solution in **C**.
2. Use **pointers** to swap characters.
3. The solution should ideally have a time complexity of $O(n)$ and a space complexity of $O(1)$.

## 🛠️ Strategy Hint
1. Reverse the entire string first.
2. Iterate through the reversed string and reverse each individual word to restore its original character order.
